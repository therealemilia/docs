# Server beitreten

Auf dieser Seite erfährst du, wie du dem **Nordverkehr FiveM Server** beitreten kannst.

Der einfachste Weg führt über unsere Connect-Seite.

## Schnell verbinden

Der einfachste Weg, dem Server beizutreten, ist über folgenden Link:

<Card title="Direkt verbinden" icon="plug" horizontal href="https://nordverkehr.xyz/connect">
  Öffnet FiveM und verbindet dich direkt mit Nordverkehr.
</Card>

## Alternative Verbindungsmöglichkeiten

Falls der direkte Link nicht funktioniert, kannst du dich auch manuell verbinden.

### Über den FiveM Server Browser

1. Öffne **FiveM**
2. Gehe auf **Play**
3. Suche nach **Nordverkehr**
4. Wähle den Server aus
5. Klicke auf **Connect**

### Über die FiveM-Konsole

Du kannst dich auch über die FiveM-Konsole verbinden.

Öffne dafür die Konsole in FiveM und gib folgenden Befehl ein:

```txt
connect connect.nordverkehr.xyz
```

Danach solltest du direkt mit dem Server verbunden werden.

## Voraussetzungen

Bevor du dem Server beitrittst, solltest du sicherstellen, dass:

- FiveM installiert ist
- GTA V korrekt eingerichtet ist
- du Zugriff auf den Server hast
- dein Voicechat eingerichtet ist

<Warning>
  Voicechat ist auf Nordverkehr erforderlich. Ohne funktionierenden Voicechat ist reguläres Roleplay nicht möglich.
</Warning>

## Probleme beim Verbinden?

Falls du dem Server nicht beitreten kannst, prüfe zuerst:

- ob FiveM gestartet ist
- ob du mit dem richtigen Account angemeldet bist
- ob du Zugriff auf den Server hast
- ob der Server aktuell online ist
- ob dein Voicechat korrekt eingerichtet ist
